שירות התמלול החופשי זמנית לא זמין

הורדנו את השרת בגלל שהצריכה עברה את היכולת התקציבית שלנו...

השארנו את אליעזר הבוט בווטסאפ ומתלבטים כיצד להמשיך.

אנחנו כמובן שמחים שאתם מוצאים את מודל התמלול שלנו מדיבור לטקסט שימושי לצרכיכם, ונשמח להמשיך ולעמוד לשירותכם כדי להפוך את הדיבור בעברית לכתוב בצורה מדויקת.

נציין בהזדמנות זו שכדי לעשות את זה אנחנו זקוקים לכסף...

כי את הנתונים, התיוג, השרתים לאימון, הליווי המשפטי ועוד קיבלנו ממתנדבים ותרומות, אבל השימוש בשרת התמלול צורך שימוש נוסף בשרתים, שעולה כסף.
